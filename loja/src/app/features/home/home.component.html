Integração com a API La Couro

Faça login para que o interceptor injete automaticamente o token nas chamadas e depois consuma os serviços de usuários, produtos, pedidos e estoque.

Refazer login Abrir Swagger
  • Auth: /api/auth/login e /api/auth/register
  • Usuários: /api/usuarios (listagem, admin, desativação)
  • Produtos: /api/produtos (CRUD e visibilidade)
  • Estoque: /api/estoque/entrada|saida
  • Pedidos: /api/pedidos (criar, pagar, cancelar)
